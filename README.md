# Introducción a Python Científico

**Fundamentos, manejo de datos y visualización**

Manual práctico para aprender Python desde cero aplicado a ciencias. No se asume conocimiento previo de programación. Cada capítulo incluye ejemplos ejecutables en Google Colab sin instalar nada.

Sitio web: [tomvalderrama.github.io/manual-introduccion-python-cientifico](https://tomvalderrama.github.io/manual-introduccion-python-cientifico/)

## Contenidos

| Parte | Capítulos |
|---|---|
| I — Fundamentos | Entorno de trabajo, sintaxis, control de flujo, funciones, debugging |
| II — Manejo de datos | NumPy, Pandas, lectura de archivos (CSV, Excel, NetCDF) |
| III — Visualización | Matplotlib, rosas de viento, figuras para informes, gráficos interactivos |

## Uso en Colab

Cada capítulo tiene un badge **Open in Colab** para ejecutar los ejemplos directamente en el navegador.

Los notebooks están en la carpeta [`notebooks/`](notebooks/).

## Uso local

```bash
pip install numpy pandas matplotlib scipy openpyxl windrose plotly
```

Clonar y abrir el notebook del capítulo de interés con Jupyter.
