# Introducción a Python Científico

**Fundamentos, manejo de datos y visualización**

Manual práctico para aprender Python desde cero aplicado a ciencias. No se asume conocimiento previo de programación. Cada capítulo incluye explicaciones conceptuales y ejemplos de código que se pueden ejecutar directamente en Google Colab sin instalar nada.

## Contenidos

- **Parte I** — Entorno de trabajo, sintaxis y fundamentos del lenguaje
- **Parte II** — Manejo de datos con NumPy, Pandas y lectura de archivos
- **Parte III** — Visualización con Matplotlib, rosas de viento y gráficos interactivos

## Cómo usar este manual

Cada capítulo tiene un botón **Abrir en Colab** que permite ejecutar los ejemplos en el navegador sin instalar Python. Para trabajar localmente:

```bash
pip install numpy pandas matplotlib scipy openpyxl windrose
```

## ¿Qué sigue?

Este manual cubre los fundamentos. Para análisis más específicos — estadísticas circulares, FFT, automatización de informes, datos geoespaciales, modelos oceánicos y datos satelitales — ver el [Manual Avanzado](https://tomvalderrama.github.io/manual-python-oceanografia/).
